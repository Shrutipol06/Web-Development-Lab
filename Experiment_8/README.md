# Experiment: HTML Table Tag

## Aim

To study and implement different types of HTML table tags to display data in tabular form on a web page.

## Objective

- To understand the HTML table structure
- To learn how to create rows and columns
- To understand tags like table, tr, th, and td
- To apply attributes like border, cellpadding, and cellspacing
- To design structured data using tables


## Types of Table Usage

### 1. Basic Table

Creates a simple table using table, tr, and td tags.

### 2. Table with Header

Uses th tag to define table headings.

### 3. Table with Border

Applies border attribute to make table visible.

### 4. Table with Cell Padding and Spacing

Uses cellpadding and cellspacing to adjust space inside and between cells.

### 5. Merged Cells Table

Uses rowspan and colspan to merge rows and columns.


## Features of Table Tag

- Displays data in structured rows and columns
- Supports headings and data cells
- Allows merging of cells
- Provides spacing and border customization
- Helps in organizing information clearly


## Learning Outcome

- Learned how to create tables using HTML
- Understood table structure and tags
- Learned to use headers and merge cells
- Gained knowledge of formatting tables
- Improved webpage layout design skills

# Experiment: HTML Link Tag

## Aim

To study and implement different types of HTML link (anchor) tags to create hyperlinks on a web page.

## Objective

- To understand the HTML anchor (a) tag
- To learn how to create links between web pages
- To understand attributes like href, target, and title
- To create internal and external links
- To apply linking techniques in web pages


## Types of Link Usage

### 1. Basic Link

Creates a simple hyperlink using the href attribute.

### 2. Link with Target Attribute

Uses target attribute to open link in a new tab or same window.

### 3. Internal Link

Links one page to another page within the same website.

### 4. External Link

Links to a webpage on another website using full URL.

### 5. Image as a Link

An image can be used as a clickable link.


## Features of Link Tag

- Creates hyperlinks between pages
- Supports navigation within and outside websites
- Allows opening links in new tabs or windows
- Can be applied to text and images
- Improves website connectivity and usability


## Learning Outcome

- Learned how to create hyperlinks using HTML
- Understood anchor tag and its attributes
- Learned internal and external linking
- Gained knowledge of navigation between web pages
- Improved basic web designing skills

# Experiment: HTML Form Tag

## Aim

To study and implement HTML form tags to collect user input on a web page.

## Objective

- To understand the HTML form tag
- To learn different input types
- To understand attributes like action and method
- To create user input forms
- To apply form elements in web pages


## Types of Form Usage

### 1. Basic Form

Creates a simple form using form tag.

### 2. Text Input Fields

Uses input type="text" for entering text data.

### 3. Password Field

Uses input type="password" to hide user input.

### 4. Radio Buttons

Allows selection of only one option from multiple choices.

### 5. Checkboxes

Allows selection of multiple options.

### 6. Submit and Reset Buttons

Used to submit form data and reset the form.

### 7. Dropdown List

Uses select and option tags to create a list of options.


## Features of Form Tag

- Collects user input
- Supports multiple input types
- Allows data submission to server
- Provides interactive elements
- Improves user interaction


## Learning Outcome

- Learned how to create forms using HTML
- Understood different input types
- Learned to use buttons and selection fields
- Gained knowledge of user data collection
- Improved webpage interactivity

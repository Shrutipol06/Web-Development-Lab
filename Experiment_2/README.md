# Experiment: Static Web Page using HTML

## Aim
To create a simple static web page using basic HTML tags and display its output in a web browser.

## Description
This experiment demonstrates the use of basic HTML elements such as headings, paragraphs, lists, images, and links to design a static web page. The page is created using HTML and does not include any dynamic functionality.

## Files Included
- `static_webpage.html` – HTML file containing the static web page code
- `output/` – Folder containing the screenshot of the output

# Experiment: HTML Text Formatting Tags

## Aim
To study and understand HTML text formatting tags used to style and display content on a web page.

---

## Objective
- To learn basic HTML formatting tags  
- To understand how text appearance can be changed  
- To identify commonly used text styling tags  

---

## Software Required
- Windows Operating System  
- Notepad / VS Code  
- Any Web Browser  

---

## Formatting Tags Used

| Tag | Description |
|-----|-------------|
| <b> | Displays bold text |
| <i> | Displays italic text |
| <u> | Underlines the text |
| <strong> | Indicates important text |
| <em> | Emphasized text |
| <mark> | Highlights text |
| <small> | Displays smaller text |
| <sup> | Superscript text |
| <sub> | Subscript text |
| <del> | Shows deleted text |

---

## Output
Text is displayed in various formats such as bold, italic, underline, highlighted, superscript, and subscript.

---

## Advantages of Formatting Tags
- Improves readability  
- Highlights important content  
- Makes web pages attractive  
- Helps organize information  

---

## Conclusion
HTML text formatting tags help enhance the appearance of web content. These tags were studied successfully and understood.

---

## Learning Outcome
- Learned different text formatting tags  
- Understood their purpose  
- Gained basic knowledge of HTML styling

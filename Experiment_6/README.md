# Experiment: HTML Frames

## Aim

To study and implement different types of HTML frames (frameset and frame) to divide a web page into multiple sections.

## Objective

- To understand the concept of HTML frames
- To learn how to divide a webpage using frames
- To understand frameset, frame, and noframes tags
- To create layouts using rows and columns
- To load different HTML pages into different sections


## Types of Frame Usage

### 1. Basic Frameset

Divides the browser window into multiple sections using frameset.

### 2. Frameset with Rows

Divides the page horizontally into rows.

### 3. Frameset with Columns

Divides the page vertically into columns.

### 4. Nested Frameset

Frames inside another frameset to create complex layouts.

### 5. Frame with Navigation

One frame is used for menu/navigation and another for displaying content.


## Features of Frames

- Divides a webpage into multiple sections
- Loads different HTML pages in each frame
- Supports both row-wise and column-wise division
- Allows reuse of navigation menus
- Helps in creating structured layouts


## Learning Outcome

- Learned how to create frames using HTML
- Understood frameset and frame tags
- Learned to design layouts using rows and columns
- Gained knowledge of loading multiple pages in a single window
- Understood basic webpage structuring using frames

# Experiment: Resume Webpage using HTML and CSS

## Aim

To design and implement a resume webpage using HTML for structure and CSS for styling.

## Objective

- To understand webpage structure using HTML
- To learn styling using CSS
- To create a professional resume layout
- To apply text formatting and layout design
- To combine HTML and CSS for better presentation


## Types of Resume Webpage Sections

### 1. Personal Information

Displays name, contact details, and basic information.

### 2. Career Objective

Shows the goal or objective of the candidate.

### 3. Educational Details

Lists academic qualifications in structured format.

### 4. Skills Section

Displays technical and soft skills.

### 5. Projects Section

Includes details of projects completed.

### 6. Experience Section

Shows work or internship experience.

### 7. Hobbies and Interests

Displays personal interests.

### 8. Declaration

Includes a statement confirming the correctness of information.


## Features of Resume Webpage

- Structured and professional layout
- Styled using CSS for better appearance
- Easy to read and organized content
- Responsive and visually appealing design
- Combines content and design effectively


## Learning Outcome

- Learned how to create structured webpages using HTML
- Understood styling using CSS
- Learned layout design for professional resumes
- Gained knowledge of combining HTML and CSS
- Improved web designing and presentation skills

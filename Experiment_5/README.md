# Experiment: HTML Image Tag

## Aim

To study and implement different types of HTML image (`img`) tags to display images on a web page.

## Objective

* To understand the HTML image tag
* To learn how to display images from different sources
* To understand image attributes like `src`, `alt`, `width`, and `height`
* To apply image alignment and linking techniques


## Types of Image Usage

### 1. Basic Image Tag

Displays an image using the `src` attribute.

### 2. Image with Width and Height

Used to control image size using `width` and `height` attributes.

### 3. Image with Alternate Text

Uses `alt` attribute to display text if image cannot be loaded.

### 4. Image as a Link

An image can be used as a clickable link using anchor (`a`) tag.

### 5. Image from Online Source

Displays images using external URLs.

## Features of Image Tag

* Displays images on web pages
* Supports size customization
* Provides alternate text for accessibility
* Allows linking images
* Supports local and online images




## Learning Outcome

* Learned HTML image tag usage
* Understood image attributes
* Learned how to link and resize images
* Gained basic webpage design knowledge

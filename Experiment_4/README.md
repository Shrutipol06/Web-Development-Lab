# Experiment: HTML List Tags

## Aim
To study and implement HTML list tags to display information in an organized manner.

---

## Objective
- To understand ordered and unordered lists  
- To learn nested list concepts  
- To display different list styles  
- To organize content using lists  

---

## Software Required
- Windows Operating System  
- Notepad / VS Code  
- Any Web Browser (Chrome / Edge)

---

## Types of Lists Used

### 1. Ordered List
Displays items in numbered format.

### 2. Unordered List
Displays items with bullet points such as disc, circle, and square.

### 3. Nested List
A list inside another list used to show sub-items.

---

## Features of List Tags
- Helps arrange data clearly  
- Improves readability  
- Supports multiple bullet styles  
- Allows multi-level listing  

---

## Output
The web page displays ordered lists, unordered lists with different shapes, and nested lists for structured presentation.

---

## Advantages
- Easy content organization  
- Clean layout  
- Beginner friendly  
- Widely used in web design  

---

## Conclusion
HTML list tags were successfully implemented to create structured and nested lists. This helps in presenting information clearly on web pages.

---

## Learning Outcome
- Learned different HTML list types  
- Understood nested list structure  
- Gained basic webpage design knowledge
